### RasterAnalysis
> 基于 PyQGIS 的栅格分析软件

#### 开发环境
QGIS 3.1、Python 3.7、PyQt5

#### 软件截图
![DEM渲染](https://lsq210.github.io/photo-album/readme/RasterAnalysis/render.jpg)
![K](https://lsq210.github.io/photo-album/readme/RasterAnalysis/k-means.jpg)

#### 功能
- 地图基本操作
- 加载栅格、矢量影像
- 图层选择
- 栅格值直方图
- 夜光遥感分析
- 多光谱影像分析
  - K 均值聚类
  - 计算植被指数
- DEM 渲染
  - 单波段灰色法
  - 伪彩色算法
  - 山体阴影法
- 调用 WFS、WMS
- 多种主题切换
- 自适应布局
